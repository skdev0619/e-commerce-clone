# E-commerce

### 1. 개요
이 프로젝트는 **전자상거래(e-commerce)의 핵심 기능**을 간소화하여 구현한 학습용 프로젝트입니다.  
**잔액 관리, 상품, 주문, 결제, 선착순 쿠폰 발급, 인기 상품 조회** 등 필수적인 기능을 중심으로 설계하였습니다.
